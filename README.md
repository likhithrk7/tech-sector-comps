## 📚 Finance Concepts Demonstrated

- **Comparable Company Analysis (Comps)** — relative valuation methodology
- **Enterprise Value (EV)** — total business value = Market Cap + Debt − Cash
- **EV/Revenue** — valuation relative to top-line revenue
- **EV/EBITDA** — most widely used M&A and LBO valuation multiple
- **P/E Ratio** — price relative to earnings per share
- **Peer Group Statistics** — mean, median, high, low analysis
- **Implied Valuation** — applying multiples to derive fair value
